Spring Pet Clinic Application

이 프로젝트는 AWS 클라우드 환경에서 확장 가능하고 안정적인 3계층 아키텍처를 구현한 Spring Boot 기반의 Pet Clinic 애플리케이션입니다. Auto Scaling, Load Balancing, 그리고 CloudWatch 모니터링을 통해 높은 가용성과 성능을 제공합니다.

3-Tier Architecture

Web, WAS, Database 계층으로 분리된 확장 가능한 아키텍처입니다. Nginx 웹 서버, Tomcat WAS, 그리고 RDS MySQL로 구성되어 있습니다.

Auto Scaling

트래픽에 따라 자동으로 인스턴스를 확장/축소하여 비용 효율성과 가용성을 보장합니다.

Load Balancing

Application Load Balancer를 통해 트래픽을 분산하고 고가용성을 제공합니다.

CloudWatch

실시간 모니터링과 알람을 통해 시스템 상태를 지속적으로 관찰합니다.

RDS MySQL

관리형 데이터베이스로 자동 백업, 패치 관리, 높은 안정성을 제공합니다.
